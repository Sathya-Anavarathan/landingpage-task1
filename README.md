# Trabook - Travel Landing Page

A modern, responsive travel booking landing page built with HTML and CSS.

## Features

- **Hero Section** - Eye-catching banner with call-to-action
- **Search Bar** - Location, date, and guest selection
- **Services Section** - Sign up, pricing, and travel features
- **Deals & Discounts** - Exclusive travel offers with pricing
- **Vacation Plans** - Popular destination packages
- **Testimonials** - Customer reviews and feedback
- **Blog Section** - Latest travel articles
- **Newsletter Signup** - Email subscription
